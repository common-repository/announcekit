Beautifully designed newsfeed powered with fancy announcement widgets

== Description ==
AnnounceKit is a beautifully designed newsfeed powered with fancy announcement widgets. It's a best way to announce news, product updates, release notes, changelog or whatever you call it.

With this plugin, you can quickly add AnnounceKit widgets into your Wordpress site.

Available announcement widget types:
- Announcement bar 
- Announcement modal 
- Line widget 
- Badge wiget 
- Floating Button 
- in-page Embeded 
- Sidebar widget 
- Popup widget [

[Check all samples](https://announcekit.app/widgets)
Register [AnnounceKit](https://announcekit.app/?ref=wp) for free and create your newsfeed with eye-catching announcement widgets.

== Frequently Asked Questions ==

= Can I attach widget to the menu?  =
Yes, you can attach Badge and Line widget to any menu on your Wordpress.

= Can I add multiple widgets? =
Yes, you can add several widgets on the one page.

= How can I integrate widget into my Wordpress site?
[https://announcekit.app/docs/wordpress](https://announcekit.app/docs/wordpress)

= Can I change the position of the widget?
Yes, you can set right, left, top, and bottom values on editing page to fine-tune the position of the widget.
